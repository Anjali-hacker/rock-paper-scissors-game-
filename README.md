# 🪨 Rock Paper Scissors

A simple **Rock Paper Scissors game** built using **HTML and JavaScript** as part of my learning journey with *The Odin Project*.

---

## 📌 Features

* Play Rock, Paper, Scissors against the computer
* Random computer choice using JavaScript
* Instant result display (Win / Lose / Draw)
* Simple and clean UI

---

## 🛠️ Tech Stack

* HTML
* JavaScript

---

## 📚 What I Learned

* JavaScript fundamentals (functions, variables, conditionals)
* DOM manipulation
* Event handling
* Basic game logic
